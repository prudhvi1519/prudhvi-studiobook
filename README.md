# Prudhvi StudioBook - Cinematic Portfolio

A high-performance, cinematic scrollytelling portfolio built with Next.js, Tailwind, and GSAP.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Motion**: GSAP + ScrollTrigger (Integration pending Phase 2)
- **Language**: TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Features (Phase 1)
- [x] Cinematic Dark Theme + Noise Overlay
- [x] Recruiter Mode (Motion Toggle)
- [x] Reduced Motion Support
- [x] Static Scene Shells (Hero, Projects, Experience, etc.)
- [x] Bento Grid Layouts
- [x] Drawer & Modal UI Primitives
